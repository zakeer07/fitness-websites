## Branch workflow

Before starting any new task:

1. Pull latest main into local main:
   ```
   git checkout main
   git pull origin main
   ```
2. Create a new branch off main:
   ```
   git checkout -b <descriptive-branch-name>
   ```
3. Do all work on that branch, commit, and push.
4. Never develop directly on main.
